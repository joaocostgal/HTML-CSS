no HTML as Pseudo-classes são representadas em CSS por :

    Exemplo:
        div:hover{
            background-color: yellow;
            }
    esse comando fará com que ao passar o mouse por cima de uma div, sua cor de funde se altere

modelo de div interativo

modelo de ocultação de texto dentro de uma div

    div  p{
        display: none;
        }
    div:hover > p{
        display: block;
        color: white;
        background-color: red;
        width: 300px;
        }

 feito isso, o texto oculto, aparecerá somente quando o mouse passar por cima da div
