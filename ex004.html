Apostila 003: tipo de listas

Trabalhando com litas

listas ordenadas

  1. acordar
  2. tomar café
  3. escovar os dentes

para criar uma lista ordenada utiliza-se a tag /ol e para adicionar itens a listas, utiliza-se a tag /li

Obs: /li ao final de cada item da lista ordenada é opcional

para personalizar sua lista, utilize o comando type =""

e caso queria que sua lista comece por um comando especifico utilize start =""

Listas não ordenadas

  • macarrão
  • arroz
  • batata
  • mantega

para criar uma lista não ordenada, utiliza-se a tag /ul

no caso da personalização desse modelo de lista, segue o mesmo código da lista ordenada
