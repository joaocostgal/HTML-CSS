Apostila 010: Pseudo-elementos

Para alterar as propriedades dos links basta:

        a{
            color: cor;
            text-decoration: none;
            }
        a:visited{
            color: cor2;
            }
        a:hover{
            text-decoration: underline;
            color: darkviolet;
            }
        a:active{
            color: aqua;
            }
        a::after{
            content: '[link]'
            color: lawngreen;
            }

isso deixará os link normais da cor1 e os que já foram visitados na cor2

        a:visited{
            color: cor2;

enquanto o :hover é a interatividade pela sobreposição do cursor do mouse

        a:hover{
            text-decoration: underline;
            color: darkviolet;
            }

o Pseudo-elementos :active modificará o link no memoento do clique

        a:active{
            color: aqua;
            }

Pseudo elementos devem ter duas vezes os :. Exemplo:

        a::after{
            content:' [link]';
            color: lawngreen;
            }

você pode utiliza um Pseudo-elementos em uma classe

        .classe::pseudo-elemento

porntuação para CSS

  • # = id
  • . = class
  • : = pseudo-classse
  • :: = pseudo-elemento
  • > = children(filiado)
