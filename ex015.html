Apostika 012: Variáveis em CSS

Declarando variáveis

para declarar variáveis em css, utilize a pseudo classe root

        root:{
            --cor1: black;
            --cor2: white;
        }

assim você deixará cores salvas para uso e se aplicaram em toda sua página

para utilizar a variavél em em elemento do site bata:

        h1{
            color: var(--cor1);
        }
