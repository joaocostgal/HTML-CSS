personalização de IDs

O id é um comando feito para inserir um codigo um unico elemento, assim podendo personaliza-lo de forma exclusiva

para inserir um id a um comando:

    Exemplo:
        h1 id="nome"/h1
    Após fazer isso, basta identificar o 
    campo no código em CSS
        h1#nome{}

tudo que em HTML é ID, em CSS fica como #

personalização de Classes

Uma classe pode ser utilizada para utilizar a mesma configuração para mais de um elemento

para inserir uma classe a um comando:

    Exemplo    
        h2 class="nome"/h2
    Após fazer isso, basta identificar o campo
    no código em CSS
        h2.nome{
            background-color: white;
            }

tudo que me HTML for class, em CSS se torna .

OBS: Você pode utilizar a classe em qualquer elemento textual

Você pode inserir duas classes no mesmo objeto:

        p class="nome1 nome2" TEXTO /p

Não existe a necessidade de se colocar separadores, apenas o espaço

utilize a tag /span para poder inserir uma clase em um trecho especifico do texto

Você pode utilizr um id e uma calsse em um mesmo elemento

        h1 id="nomeid" class="nomeclass" /h1

Regras de hierarquia

  • configurações genericas
    •     h1{
        color: white;
        }
  • configuraçoes personalizadas
    •     h1#nomeid{
        color: green;
        }

as configurações semprem herdam as configurações da camada superior, mas as novas se sobrepoem
