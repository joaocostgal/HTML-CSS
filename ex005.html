Usando links

Links externos

Você pode acessar links externos como o YouTube

A tag que voce utiliza para fazer o hyperlink é /a

voce pode fazer com que o hyperlink abra uma nova aba em seu navegador quando clicar no link

clique nesse link para ser direcionado para outra aba de seu navegador

para utilizar esse recurso, coloque target="_blank" e rel="external", apos inserir o link de destino
