Apostila 012: Configurações CSS

box-level

parágrafos também são exemplos de caixas box-level, mas os links são exmplos de caixas inline-level. Vamos ver como isso funciona.

comandos Css para box-level e inline-level

        
        height: 300px; = altura da box
        width: 300px; = lados da box
        background-color: azure; = cor de fundo   
            
        border = comanos para bordas
            
        border-width: 10px; = tamanho da borda da box
        border-style: solid; = estilo da borda da box
        border-color: brown; =cor da borda da box
            
        padding = comandos para espaçamento interno

        padding-top: 10px; = espaçamento do top
        padding-right: 10px; = espaçamento da direita
        padding-bottom: 5px;  =   espaçamento de baixo
        padding-left: 5px; = espaçamente da esquerda
            
        margin = comando da borda externa da box

        margin-top; 10px: margem de cima
        margin-right: 10px; margem da direita
        margin-bottom: 10px; margem de baixo
        margin-left: 10px; = margem da esquerda

        outline = comando para as linhas de fora
            
        outline-width: 10px; = largura da linha
        outline-style: solid; = estilo da linha
        outline-color: aqua; = cor da linha

OBS: o comando automatico segue a seguinte hierarquia

        Ex:
        margin: auto;
        margin-top
        margin-right
        margin-bottom
        margin-left

seguindo o sentido horário de um relógio

os comando também podem ser simplificados

            maring: 10px 15px 10px 15px black;

OBS: Caso o valor das margens sejam iguais, ele deve ser anotado somente uma vez

        margin: 10px;
        maring: 10px 20px; = neste caso o primeiro 
                             valor será referente
                             ao top e bottom e o outro
                             será right e left

a simplificação de estilos, segue a seguinte hierarquia

        border-width: 5px;
        border-style: solid;
        border-color: black;

podendo ser simplificada para

            border: 5px solid black;

utilizer o comando display para alterar o modelo da box

        display: block; = deixará a box como box-level
