grouping tags

  • header = cabeçalho
    • nav = navegação
  • main = corpo pricipal
    • section = seção
      • article = artigo
        • aside = assunto afiliado
  • footer = rodapé

Sombras

sombras é um recurso estético para o site

para inserir sombras, utilize o comando:

        box-shadow: 1 px 1px 1px black;

bordas

o comando border-radius poder ser utilizado para arredondamento de bordas

        border-radius: 5px;

    também possui o seu detalhamento

        border-top-left-radius:
        border-top-right-radius:
        border-bottom-right-radius:
        border-bottom-left-radius:

bordas decoradas

        border: 20px solid black;
        padding: 10px;
        border-image-source: link da imagem;
        border-image-slice:20;
        border-image-repeat: repeat;
        
        border-image: url('borda.png') 27 repeat;
