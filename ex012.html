Apostila 011: modelo de caixa

uma caixa é definia por duas medidas importantes:

  • altura = height
  • largura = width

a caixa também é composta por uma borda

  • border = borda
  • padding = acolchoamente(distancia interna da borda)
  • margin = margem apartir da borda do objeto
  • outline = contorno/ traçado

tipos de caixas

  • box-level (div)cria uma divisão com quebra de linha
  • inline-level (span) cria uma divião e da continuidade na mesma linha

box-level

  • div
  • h1 ~ h6
  • p
  • main
  • header
  • nav
  • article
  • asider
  • footer
  • form
  • video

inline-level

  • span
  • a
  • code
  • small
  • strong
  • em
  • sup ~ sub
  • label
  • button
  • input
  • select
