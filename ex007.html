Apostila 006: CSS

CSS é referente a linguagem visual do site, responsável pelo design

para iniciar a personalização do site você deve utilizar a tag /style e colocar os modificadores dentro dela

estilos de CSS

os códigos CSS podem ser utilizados de 3 maneiras

  1. Estilo linear: age somente na linha que está sendo excultado
  2. Estilo local: agr somente na pagina que está esndo execultado
  3. Estilo por link: age em toda pagína que estiver utilizando o link de acesso

para utilizar o link utilize a tag /link:css e modifique o href="style.css"

quando você quiser modificar um determinado elemento voce deve utilizar sua tag e utilizer os códigos CSS dentro de {}

        style
        body{
            background-color: black:
        }
        /style

assim você modificar a cor de fundo do corpo do seu site

Regras em CSS

utiliza-se o @ para iniciar uma regra

utilizer o * {} para criar um padrao em CSS para todos os demais

Códigos de edição em CSS

  • color:modificador das cores
  • background: modificador do fundo
  • text: modificador de textos
  • border: modificador para bordas
  • height: altura
  • width: largura
  • padding: acolchoamento
  • margin: margens

Divisões do site

  • head: cabeçalho do navegador
  • body: corpo principal
  • footer: rodapé da página
  • main: conteúdo principal
  • section: area de seleção
  • div: divisória
  • h1: títulos(podendo ter subdivisões como h2, h3 e etc
  • p: para parágrafo
